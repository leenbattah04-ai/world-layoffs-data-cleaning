# World Layoffs Data Cleaning Project

## Overview
This project focuses on cleaning the World Layoffs dataset using MySQL.

## Skills Demonstrated
- Data Cleaning
- Window Functions
- ROW_NUMBER()
- Self Join
- Handling Null Values
- Date Conversion
- Data Standardization

## Technologies Used
- SQL
- MySQL

## Cleaning Process

1. Create staging tables.
2. Remove duplicate records.
3. Standardize text values.
4. Handle missing values.
5. Convert dates into DATE format.
6. Remove unnecessary rows and columns.
